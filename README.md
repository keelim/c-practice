1. **포인터**
    - `pointer` 컴퓨터 상 메모리 주소
    - &을 붙여서 표현을 한다. 
    - 포인터 변수
    - 역의 관계가 성립을 하는 것이다. (&)
    - `int *mission` 포인터 변수의 선언 > 또한 주소 값을 가짐
    - `mission = &var` > 주소 값을 get
    - `*mission` > 주소에서의 변수 값을 know   
    - 2개의 포인터 변수는 메모지 주소를 refer
    - 메모리 주소에서 값을 바꾸었기 때문에 기존의 변수의 값또한 변경
    - pointer 배열의 관계    
        - *(arr + i) arr[i] 와 같은 표현
